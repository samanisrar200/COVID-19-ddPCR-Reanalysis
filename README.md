# COVID-19 ddPCR Reanalysis

## 📌 Objective
Independent reanalysis of droplet digital PCR (ddPCR) data from COVID-19 patients to demonstrate data handling, visualization, and statistical interpretation.

## 📊 Dataset
- Source: GEO (Accession: GSE150728)
- Content: ddPCR measurements of SARS-CoV-2 viral targets + patient metadata
- Format: Raw counts & fluorescence values

## 🔬 Methods & Workflow
1. Data acquisition from GEO
2. Preprocessing & normalization
3. Exploratory data analysis (EDA)
4. Statistical testing (group comparisons, correlations)
5. Visualization & summary reporting

## ✅ Expected Outcomes
- Clean, reproducible pipeline for ddPCR reanalysis
- Visualization of viral load dynamics
- Insights into patient variability & metadata associations
